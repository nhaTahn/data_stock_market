`training_runs` structure:

- `active/`
  Current runs and experiments for `VHM`, `IMP`
- `active/reports/`
  Comparison CSV/PNG for current runs
- `search_runs/`
  Batch feature-search folders such as `search_*`
- `representative_runs/`
  Representative-sector experiments
- `reports/`
  Aggregated summaries, sector rollups, shared plots

Quick places to open:

- `active/`
- `active/reports/`
- `reports/plots/`
